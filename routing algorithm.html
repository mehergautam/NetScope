🎯 Dijkstra's Algorithm
Greedy approach that finds shortest path by exploring nodes in order of increasing distance. Works only with non-negative weights. Time: O((V+E) log V) with min-heap.
⚡ Bellman-Ford Algorithm
Uses dynamic programming to find shortest paths. Can handle negative weights and detects negative cycles. Slower but more versatile. Time: O(VE).
🌐 Real-World Usage
Used in OSPF (Open Shortest Path First) and IS-IS routing protocols. GPS navigation, network packet routing, and social network analysis all use shortest path algorithms.
