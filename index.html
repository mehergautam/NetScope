About

The Person Behind NetScope

🎯 Mission

NetScope is a free, open-source learning platform for CS students and networking beginners. Most textbooks tell you what TCP does — NetScope shows you how it actually works.
No signup. No cost. Just click and learn.

MG

Meher Gautam

Computer Science Student & Developer

Built NetScope because networking concepts always felt abstract and hard to grasp from textbooks alone. Every animation here represents something I struggled to visualize the first time — so I built the tool I wished existed when I was learning.

✉️ mehergautam968@gmail.com 🐙 github.com/mehergautam 💼 linkedin.com/in/mehergautam 🌐 mehergautam.github.io/NetScope
💡
Why NetScope?
Textbooks explain what protocols do, but rarely show how they work. NetScope fills that gap — watch actual packets move, headers get added, and routes get computed in real time.
🛠️
Built With
Pure HTML, CSS & JavaScript — no frameworks, no libraries. Just clean code and hand-crafted animations. Hosted on GitHub Pages, open source on GitHub.
🚀
What's Next?
More topics coming — HTTPS handshake, DHCP, BGP deep dive, and maybe a quiz mode. Suggestions welcome via the feedback form below!
📖
Who Is This For?
CS students, networking beginners, interview preppers, and anyone who learns better by seeing things happen — not just reading about them.
