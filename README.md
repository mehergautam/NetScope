## 🎯 What is NetScope?

NetScope is a **free, browser-based Computer Networking visualizer** that transforms abstract networking concepts into beautiful, interactive animations. Instead of reading walls of text, users can *see* how data flows, how protocols work, and how networks actually function — in real time.

> 💡 **Designed for:** CS students, networking beginners, interview prep, and anyone who learns better visually.

## 🧠 Why I Built This

Learning Computer Networks from textbooks is hard. Concepts like TCP handshakes, ARP resolution, and routing algorithms are inherently *visual* — but most resources only explain them in text.

NetScope bridges that gap. Every topic has a dedicated, animated simulation that makes the concept click intuitively.
