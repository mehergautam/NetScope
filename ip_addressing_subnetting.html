🌐 IP Address Structure
An IP address consists of 32 bits divided into 4 octets. It's split into Network (identifies the network) and Host (identifies the device) portions.
🔢 Subnet Mask
Determines which portion is network vs host. Contiguous 1s represent network bits, 0s represent host bits. Example: 255.255.255.0 = /24 (24 network bits).
📐 CIDR Notation
Classless Inter-Domain Routing - compact way to specify subnet mask. /24 means first 24 bits are network. Allows flexible subnetting beyond traditional classes.
