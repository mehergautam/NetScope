The OSI (Open Systems Interconnection) model is a conceptual framework that divides network communication into 7 distinct layers, each with a specific role. Data flows down the sender's stack (adding headers) and up the receiver's stack (stripping headers). Click any layer to learn about its protocols, devices, and real-world role.
🔻 Encapsulation (Sender)
Data travels top-down through layers 7→1. Each layer wraps the data with its own header (and sometimes trailer), adding addressing and control info — until Layer 1 transmits raw bits over the medium.
🔺 Decapsulation (Receiver)
On the receiving end, data travels bottom-up from layers 1→7. Each layer reads and strips its own header, processes the relevant info, and passes the payload up — until the application receives the original data.
🌐 Real-World Example
Sending an email: SMTP (L7) composes it, TLS (L6) encrypts it, TCP (L4) segments it, IP (L3) routes it, Ethernet (L2) frames it, and Wi-Fi (L1) transmits the bits.
